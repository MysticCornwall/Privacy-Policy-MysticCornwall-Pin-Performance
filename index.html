MysticCornwall Pin Performance

MysticCornwall Pin Performance is a private, single-user analytics tool operated by the owner of the MysticCornwall Pinterest Business account. It reads the performance of our own organic Pins (impressions, saves, clicks) through the Pinterest API and stores them in a private database to build an internal performance dashboard.

The tool has no other users. It does not publish content, run ads, or access any other Pinterest account.
