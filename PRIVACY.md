# Privacy Policy — MysticCornwall Pin Performance

_Last updated: June 10, 2026_

MysticCornwall Pin Performance is an internal, single-user analytics tool used exclusively by the owner of the MysticCornwall Pinterest Business account.

- **Data collected**: organic performance metrics (impressions, saves, pin clicks, outbound clicks) of pins published by our own account, retrieved through the Pinterest API.
- **No personal data** of any other Pinterest user is collected, processed, or stored.
- **Storage**: metrics are stored in a private database accessible only to the account owner.
- **Sharing**: data is never sold, shared, or disclosed to third parties.
- **Scopes**: the app uses read-only Pinterest API scopes (`pins:read`, `boards:read`).
