Privacy Policy — MysticCornwall Pin Performance

Last updated: 16 June 2026

1. Overview

MysticCornwall Pin Performance ("the App") is a private, single-user analytics tool operated by the owner of the MysticCornwall Pinterest Business account. It is used solely to analyze the performance of our own Pinterest content. The App has no other users.

2. Data we collect

Through the Pinterest API, using read-only scopes (pins:read, boards:read), the App accesses aggregate performance metrics for Pins published by our own account — impressions, saves, pin clicks, outbound clicks and related engagement metrics — together with each Pin's title, link, board and thumbnail. The App does not collect, access, or store any data about other Pinterest users, their accounts, or their activity.

3. How we use the data

The data is used exclusively to produce an internal performance dashboard for our own Pins. It is never used for advertising, profiling, or any decision affecting other people.

4. Data protection and security

Metrics are stored in a private PostgreSQL database (Supabase) accessible only to the account owner through credentials that are never shared. API tokens and database keys are stored as encrypted secrets and are never exposed publicly. All data transfers occur over HTTPS.

5. Data retention

Data is retained only as long as needed for performance analysis and is deleted upon request. Because the App concerns only our own account, no third-party personal data is ever retained.

6. Data sharing

We do not sell, rent, share, or disclose any data to third parties. Data never leaves the private database except for the account owner's own viewing.

7. Third-party services

The App relies on the Pinterest API (data source), GitHub Actions (scheduled execution) and Supabase (storage), each used solely to operate the App for our own account.
